#  GameMarketManager

Sistema desenvolvido em Java para gerenciar contas de jogos, permitindo cadastrar, listar, pesquisar, vender contas e visualizar estatísticas.

##  Funcionalidades

- Cadastro de contas
- Listagem de contas
- Pesquisa por ID
- Venda de contas
- Estatísticas do estoque

##  Conceitos utilizados

- Programação Orientada a Objetos (POO)
- Herança
- Polimorfismo
- Abstração
- Encapsulamento
- Interfaces
- Classes Abstratas
- Records
- Exceptions Personalizadas
- Enums
- ArrayList
- HashMap
- HashSet
